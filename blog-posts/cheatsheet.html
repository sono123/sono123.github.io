Loops Comparison

By Steve Ono, Posted on July 31, 2014

Ruby For Loops

A ruby for loop executes code until a condition is met, or one for each element in an expression.

Ruby While Loops

Executes code while conditional is true. A while loop's conditional is separated from code by the reserved word do, a newline, backslash \, or a semicolon ;.

Ruby Until Loops

Executes code while conditional is false. An until statement's conditional is separated from code by the reserved word do, a newline, or a semicolon.

Javascript For Loops

Creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement executed in the loop. These optional expressions are initialization, condition, final-expression, and the code block called the statement.

Javascript For/In Loops

The for/in loop iterates through the properties of an object. The loop iterates in arbitrary order. For each distinct property, statements can be executed.

Javascript While Loops

The while loop loops through a block of code as long as a specified condition is true. Note, the condition is evaluated BEFORE executing the statement.
